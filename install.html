System Requirements

Before installing V#, ensure your system meets the following requirements:

  • Operating System: Windows, macOS, or Linux (for mac and linux you need to build project by yourself)
  • Processor: 1 GHz or faster
  • RAM: 2 GB minimum (4 GB recommended)
  • Disk Space: At least 100 MB free
  • .NET 8.0
  • Internet Connection: Required for downloading the interpreter

Installation Steps

  1. Download the V# Interpreter:

    Visit the GitHub repository and download the latest release for your operating system.

  2. Install the Interpreter:

    Follow the instructions based on your operating system:

    • Windows: Run the downloaded `install.bat` file and follow the `how_to_install.txt` instructions.
    • macOS & Linux: download source then build the project.
  3. Verify the Installation:

    Open your terminal or command prompt and run:

    vsharp --version

    You should see the version number of V# displayed. If you see an error, check the troubleshooting section below.

Troubleshooting

If you encounter any issues during installation, consider the following tips:

  • Installation Failed: Ensure you have downloaded the correct version for your OS.
  • Command Not Found: Make sure that the V# executable is included in your system's PATH variable.
  • Permissions Issues: Try running the installer as an administrator (Windows)
